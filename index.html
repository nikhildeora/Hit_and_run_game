Hit & Run

Instructions for play :

ArrowRight : Player will run

ArrowLeft : Player will run reverse

ArrowDown : Player will sit

ArrowUp : Player will Jump

ArrowRight + Enter : Player will run with boost speed

ArrowUp + Enter : Player will Jump with boost speed

Player in air + ArrowDown : Player will dive with boost speed

Instructions for get scores :

-5 :When Player hit enemy in "Normal Speed" and also loose 1 live.

+1 :When Player hit enemy in "Boost Speed".

5 lives :Player will lost if he loose all lives by hit enemies in "Normal Speed".

Note :Player want score of 40+ to win in 30 seconds.
